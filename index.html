Getting Started

How to use these prompts

No prior data science experience needed. Follow these steps in any Google Colab notebook using the built-in Gemini AI assistant.

01 💻
Open Google Colab
Go to colab.research.google.com and start a new notebook. Sign in with your Google account.
02 ✨
Open Gemini Panel
Click the Gemini star icon in the left sidebar of Colab to open the AI coding assistant panel.
03 📋
Copy Your Prompt
Find your department's project below. Click "Copy Prompt" — copy the entire prompt text exactly.
04 🚀
Paste & Generate
Paste the prompt into the Gemini input box and press Enter. Gemini will generate all code cells for you.
05 ▶️
Run the Notebook
Click "Insert" to add generated cells, then run each with Shift+Enter in sequence.
06 🔧
Fix Errors with AI
If a cell fails, ask Gemini: "Fix the error in the previous code cell" — it will debug automatically.

Pro Tips

Getting More from AI Copilots

Use these techniques with Gemini in Colab, VS Code Copilot, or any AI assistant to go further.

🎯
State your background first
Always begin your prompt by identifying yourself. The AI calibrates its explanations to your level.
"I am a Botany MSc student with basic Python..."
🔗
Give exact data sources
Paste the exact API URL or dataset name. This prevents the AI from hallucinating nonexistent data sources.
"Download from: api.gbif.org/v1/occurrence/search"
🐛
Debug with context
When pasting an error, include the full traceback. The AI can pinpoint the fix much faster with the complete error message.
"Fix this error: [paste full traceback here]"
📦
Fix failing installs
If a pip install fails in Colab, ask specifically for the Colab-compatible install method.
"!pip install rdkit-pypi --quiet"
📊
Ask for interpretation
After generating plots, ask Gemini to explain what the visualization reveals in the context of your subject domain.
"Explain what this NDVI heatmap shows ecologically"
🔄
Iterate step by step
If a full prompt is overwhelming, break it into parts. Run Part 1 first, then continue with the next numbered section.
"Now implement only Part 2 of the previous prompt"
💡
Ask for alternatives
If a library doesn't work, ask for an alternative approach. Gemini can often suggest a simpler method.
"Suggest a simpler alternative to rdkit for this task"
📝
Request markdown notes
Ask Gemini to add markdown cells with theory explanations after each code block to create a self-documenting notebook.
"Add a markdown cell explaining the biology behind this"
